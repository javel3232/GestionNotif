# Gestión de Notificaciones

Sistema de gestión de notificaciones multiplataforma implementando el patrón Bridge.

## Instalación

```bash
npm install
```

## Ejecución

### Modo desarrollo
```bash
npm run dev
```

### Compilar y ejecutar
```bash
npm run build
npm start
```

## Comandos disponibles

- `npm run build` - Compila el proyecto TypeScript
- `npm start` - Ejecuta el proyecto compilado
- `npm run dev` - Ejecuta en modo desarrollo con ts-node
